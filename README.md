# Melexis Compact Master 81349 (MCM-81349) PCB

This project assembles the Melexis Compact Master 81349 (MCM-81349) PCB which can be used to bootload and
control the [Melexis](https://www.melexis.com) generation 4 motor driver products.

The MCM-81349 is a hardware platform which is built upon the [Espressif](https://www.espressif.com/) ESP32 and provides
an easy to use evaluation setup for the MLX81349 embedded motor driver IC.

<img src="fabrication/v1_0/MCM-81349-top.png" width="30%">

To explore the schematic and layout one can clone this repository and install KiCad or use [KiCanvas](https://kicanvas.org/?github=https%3A%2F%2Fgithub.com%2Fmelexis%2Fmcm-81349-pcb).
